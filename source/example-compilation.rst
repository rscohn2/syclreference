Example Compilation
===================


The oneAPI tools are available in several convenient forms, as detailed
in `oneAPI Toolkit
Distribution <oneapi-toolkit-distribution.html>`__ earlier
in this guide. Follow the instructions in the `Installation
Guide <https://software.intel.com/en-us/articles/installation-guide-for-intel-oneapi-toolkits>`__
to obtain and install the tools. Once the tools are installed and the
environment is set, compile code for execution.
