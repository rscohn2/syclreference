Software Development Process
============================


The software development process using the oneAPI programming model is
based upon standard development processes. Since the programming model
pertains to employing an accelerator to improve performance, this
chapter details steps specific to that activity. These include:


-  The performance tuning cycle
-  Debugging of code
-  Migrating code that targets other accelerators
-  Composability of code
